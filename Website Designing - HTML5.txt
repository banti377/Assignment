Que 1 : Are the HTML tags and elements the same thing? 
Ans 1 :

HTML tags and elements are closely related but have slightly different meanings.
HTML tags are used to hold the HTML element.
HTML element holds the content.
HTML tag starts with < and ends with > Whatever written within a HTML tag are HTML elements.

Que 3 : What are void elements in HTML? 
Ans 3 :

A void element is an element whose content model never allows it to have contents under any circumstances.
Void elements can have attributes.
The following is a complete list of the void elements in HTML : area , base , br , col , command , embed , hr , img , input , keygen , link , meta , param , source , track , wbr.


Que 4 : What are HTML Entities?
Ans 4 :

An HTML entity is a piece of text ("string") that begins with an ampersand ( & ) and ends with a semicolon ( ; ).
Entities are frequently used to display reserved characters (which would otherwise be interpreted as HTML code), and invisible characters (like non-breaking spaces).

Que 9 : How is Cell Padding different from Cell Spacing?
Ans 9 :

Cellpadding basically defines the space present between a table cell's border and the content present in it.
Cellspacing basically defines the space present between individual adjacent cells.


Que 10 : How can we club two or more rows or columns into a single row or column in an HTML table? 
Ans 10 :

It can be done by using the rowspan and colspan attribute in HTML.
The rowspan is used to merge or combine the number of cells in a row whereas the colspan is used to merge column cells in a table.


Que 11 : What is the difference between a block-level element and an inline element? 
Ans 11 :

Block elements always start from a new line. 
Block elements cover space from left to right as far as it can go.

Inline elements never start from a new line.
Inline elements only cover the space as bounded by the tags in the HTML element.

It essentially puts another webpage within the parent page.

Que 19 : What is difference between HTML and XHTML?
Ans 19 :

HTML is the standard markup language for creating web pages.
In HTML closing tag can be omitted if required.
Tags are used for header, footer, section, article, and ​nav, thus making it easier to write and read code.

XHTML is a stricter and more standardized version of HTML. 
In XHTML every element should have the corresponding ending tag.
No tags are used for header, footer, section, article, nav, and divs with classes; instead, ids have to be used.


Que 20 : What are logical and physical tags in HTML?
Ans 20 :

Logical and Physical tags are used in HTML for better visibility and understanding of the text by the user on the web page. 
However, both tags differ from each other as suggested by their names. 
Logical Tags are used in HTML to display the text according to the logical styles.
Physical Tags are used in HTML to provide actual physical formatting to the text.
